## Installation

Vous pouvez installer le package via composer :

```bash
composer require blitz-php/tasks
```

## Utilisation

```php
$skeleton = new BlitzPHP\\Tasks\Tasks();
echo $skeleton->echoPhrase('Hello, BlitzPHP\\Tasks!');
```
